Cifras

Ao invés de usar a partitura tradicional (com pentagrama e notas musicais), as cifras usam letras do alfabeto para indicar os acordes de uma música.

 Campo Harmonico

O campo harmônico é um conjunto de acordes que são formados a partir das notas de uma determinada escala musical(maior, menor, etc), seguindo uma ordem lógica. Ele é uma ferramenta muito útil para compositores e músicos, pois mostra quais acordes “combinam” entre si e podem ser usados juntos de maneira harmoniosa dentro de uma tonalidade.
